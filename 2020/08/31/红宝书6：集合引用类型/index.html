红宝书6：集合引用类型

2020-08-31 JavaScript高级程序设计 评论

Object 类型

大多数对象都是 Object 类型的实例

  • new 操作符
  • 对象字面量
    • 对象定义的一种简写形式，目的在于简化创建包含大量属性的对象的过程。

      通过对象字面量定义对象时，实际上不会调用 Object 构造函数

表达式上下文指的是期待返回值的上下文{}。

语句上下文表示一个语句块的开始。

例如let obj = {}，赋值号后面期待返回值，则为表达式上下文

if () {}，条件语句后面，则是语句上下文

Array 类型

和其他语言的区别：每一项可以保存任何类型的数据，互相之间不影响

创建数组

  • new
  • 省略 new
  • 数组字面量

    数组字面量表示法时不会调用 Array 构造函数

  • Array.from
    • 类数组结构 -> 数组实例
  • Array.of()
    • 参数 -> 数组实例

数组索引

  • length 可修改，会移除超出的项
  • 当把一个值放在超出当前数组大小的位置上时，会重新计算长度，长度值为最后一项的索引 + 1

    检测数组

value instanceof Array 假定只有一个全局执行环境，如果从一个框架向另一个框架传入数组，就具有不同的构造函数
Array.isArray()

迭代器方法

  • keys
  • values
  • entries

复制和填充方法

  • fill() 填充

  • copyWithin() 浅复制

    • 改变原数组会同步改变

    • let a = [1, 2, 3]
b = a.copyWithin()
a[2] = [1]
// a = [1, 2, 1]
// b = [1, 2, 1]

转换方法

  • Array.toString()
  • Array.toLocaleString()
  • Array.valueOf()

    栈/队列方法

  • Array.push 末尾添加
  • Array.pop 取最后一项并移除
  • Array.shift 取第一项并移除
  • Array.unshift 头部添加

    排序方法

  • Array.reverse() 反转顺序
  • Array.sort() 先转 toString()，字符串升序
    • sort 接收比较函数
      • params1 -> params2 ===> -1
      • params2 -> params1 ===> 1

        操作方法

  • Array.concat
  • Array.slice(a, b) [a, b)
    • 负数则从末尾开始数
  • Array.splice(a, b, ...c)
    • a 位置
    • b 要删除的项数
    • c 插入的内容（若干）
    • return 被删除的内容

位置方法

  • 严格相等
    • Array.indexOf() 返回下标
    • Array.lastIndexOf()
    • Array.includes() 返回布尔
  • 断言函数 (element, index, array) => {}
    • find((element, index, array) => {})
    • findIndex((element, index, array) => {})

迭代方法

  • 两个参数
    • 要在每一项上运行的函数
      • 三个参数
        • 数组项的值
        • 该项再数组中的位置
        • 数组对象本身
    • （可选的）运行该函数的作用域对象
      • 影响this的值
  • Array.every() 每一项都返回 true，则返回 true
  • Array.some() 任一项返回 true，则返回 true
  • Array.filter() 返回该函数返回 true 的项组成的数组
  • Array.map() 返回每次函数调用的结果组成的数组
  • Array.forEach() 没有返回值

以上方法都不会修改数组中包含的值

归并方法

  • Array.reduce()

  • Array.reduceRight()

  • 两个参数

    • 每一项上调用的函数
      • 四个参数
        • 前一个值
        • 当前值
        • 项的索引
        • 数组对象
    • （可选的）作为归并基础的初始值

  • 返回值作为第一个参数传给下一项

  • 第一次迭代发生在第二项上，第一个参数是数组的第一项，第二个参数是数组的第二项

Map

基本 API

  • consg m = new Map()
  • m.set()
  • m.get()
  • m.has()
  • m.size()
  • m.delete()
  • m.clear()

顺序与迭代

  • Object 无序，Map 有序
  • keys()
  • values()
  • entries()

Set

基本 API

  • const s = new Set()
  • s.add()
  • s.has()
  • s.size()
  • s.delete()
  • s.clear

顺序与迭代

  • Set 维护插入时顺序
  • keys()
  • values()
  • entries()

迭代与扩展操作

  • 有4种原生集合类型定义了默认迭代器：

    • Array
    • 所有定型数组
    • Map
    • Set

  • 特性

    • 都可以传入for-of循环
    • 都兼容扩展操作符
